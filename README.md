## О проекте
Web api для проекта Wow Ruby.
Wow ruby - это проект, нацеленный на помощь игрокам в игре World of Warcraft. Проект состоит из двух частей: база данных с предметами, НИП(неигровой персонаж) и квестами; а так же специальный алгоритм, строющий
оптимальный маршрут прокачки. Под маршрутом понимается последовательность из локаций и квестов. Эта последовательность отличается своей оптимизацией: минимальные временные потери со стороны игрока. Для создания маршрута системе необходима конфигурация: пользователь выбирает рассу, класс, желаемый уровень и прочее.

# Этап разработки
На данный момент в проекте реализованы все основные методы для работы базы данных, авторизация и аутенфикация, сама база данных.

Следующий этап это реализация непсоредственного алгоритма.

В ближайшем будущем планируется учет снаряжения в маршруте.

## Клонирование репозитория

Сначала клонируем проект и переключаемся на ветку `dev`:

```bash
git clone git@gitlab.com:plebstomsk/wow-ruby.server.git
cd wow-ruby.server
git checkout dev
git pull origin dev
```

В myapp обновляем сваггер
```
swag init -g cmd/main.go
```

Собираем Docker образы сервера и базы данных:
```
docker compose -f testing.docker-compose.yml build --no-cache
```

Поднимаем контейнеры:
```
docker compose -f testing.docker-compose.yml up -d
```
